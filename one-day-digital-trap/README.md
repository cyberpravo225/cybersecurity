# Один день: цифровая ловушка

Современная браузерная игра-обучение по кибербезопасности для новичков.

## Стек
- React + Vite
- Tailwind CSS
- Framer Motion
- WebAudio (синтез звуков, без бинарных файлов)

## Запуск
```bash
cd one-day-digital-trap
npm install
npm run dev
```

## Production
```bash
npm run build
npm run preview
```
