Обучающий материал

Подробная шпаргалка: как переводить символы в ASCII, почему используется 8-битная запись и как быстро проверять корректность двоичного ответа.

🔤

Как переводить символы в ASCII

1) Возьми символ, например A.
2) Найди его код в ASCII: A = 65.
3) Переведи число 65 в двоичный вид: 01000001.
4) Для слова переводи каждый символ отдельно: CAT → 01000011 01000001 01010100.

8️⃣

Почему используют 8 бит

Один символ в базовой таблице ASCII хранится в одном байте, а байт — это 8 бит. Поэтому запись всегда дополняют ведущими нулями до 8 символов: 1000001 становится 01000001.

🔁

Как переводить из двоичного кода обратно в текст

1) Разбей строку на блоки по 8 бит.
2) Каждый блок переведи в десятичное число.
3) По таблице ASCII определи символ.
Пример: 01001000 01101001 → 72 105 → Hi.

✅

Быстрая проверка ответа

• Проверь, что каждый символ записан ровно 8 битами.
• Сравни длину: количество байтов должно совпадать с количеством символов.
• Убедись, что в коде только 0 и 1.
• Прогони обратный перевод: если получилось исходное слово, решение верное.

Мини-шпаргалка для практики

Часто используемые буквы:
A = 01000001, B = 01000010, C = 01000011,
a = 01100001, b = 01100010, c = 01100011,
0 = 00110000, 1 = 00110001, пробел = 00100000.
