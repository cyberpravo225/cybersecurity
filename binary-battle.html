⚔️ Binary Battle

Практикуй перевод слов в двоичный код в 4 режимах: учебный, соревновательный, тестовый и турнирный.

1) Учебный режим

Без ограничений по времени. Можно запросить подсказку: ASCII-код одного символа слова.

—

2) Соревновательный режим (Timed)

60 секунд на максимум правильных переводов. За каждый правильный ответ +1 очко.

Осталось времени

60

Очки

0

Нажми «Старт»

3) Тестовый режим

Выбери количество слов (5–20). Подсказки отключены.

Текущий прогресс

0 / 0

Верно

0

Нажми «Начать тест»

4) Турнирный режим

Настрой параметры турнира: участников, сложность, время на слово, слов в туре.

Сначала создай турнир

5) Обучающий материал

Как переводить слово в двоичный код:

  1. Раздели слово на символы: cyber → c y b e r.
  2. Найди ASCII-код каждого символа: c = 99, y = 121, ...
  3. Переведи каждый код в 8-битный бинарный формат: 99 = 01100011.
  4. Склей блоки через пробел. Для cyber: 01100011 01111001 01100010 01100101 01110010

Совет: всегда проверяй, что каждый символ записан именно 8 битами.
